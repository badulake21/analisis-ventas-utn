
# Analisis de Ventas UTN

## Integrante

* Leandro Campi

## Materia

Organización Empresarial - Tecnicatura Universitaria en Programación (UTN)

## Escenario Elegido

Escenario B - Análisis de Ventas de una Pequeña Empresa.

## Objetivo

Analizar un conjunto de datos de ventas para obtener indicadores básicos de desempeño comercial.

## Estructura del Proyecto

analisis-ventas-utn/

├── datos/

├── scripts/

├── resultados/

├── README.md

└── .gitignore

## Dataset Utilizado

Archivo CSV con registros simulados de ventas que incluye:

* Fecha de venta
* Producto
* Cantidad vendida
* Monto de venta

## Indicadores Calculados

* Ventas totales
* Producto más vendido
* Ventas por mes

## Resultados Generados

El script genera:

* Resumen estadístico en consola
* Gráfico de evolución de ventas
* Archivo gráfico almacenado en la carpeta resultados

## Ejecución

python scripts/analisis_ventas.py
